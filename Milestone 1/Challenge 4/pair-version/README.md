# Pair Version — Task Manager

This version of the task manager app was built using an AI pair programming assistant where I wrote the code and used AI suggestions inline.

**Tool used:** GitHub Copilot  
**Time to build:** ~35 minutes  
**Suggestions accepted:** ~12  
**Suggestions rejected:** ~5  
**Live URL:** https://your-pair-link.netlify.app  

---

## Notes

- I built the application step-by-step and used Copilot for inline suggestions.
- I accepted suggestions for repetitive logic like filtering and UI updates.
- I rejected suggestions when they introduced unnecessary complexity or incorrect logic.
- I had full control over component structure and function implementation.
- Debugging was easier since I understood every part of the code.
- The final code is clean, modular, and easy to modify.

---

## Key Takeaway

This approach gave me full control and understanding of the code, making it easier to debug, maintain, and extend.
