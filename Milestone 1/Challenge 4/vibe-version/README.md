# Vibe Version — Task Manager

This version of the task manager app was generated using a vibe coding tool based on a single prompt.

**Tool used:** v0 by Vercel  
**Time to generate:** ~10 minutes  

**Prompt used:**
Build a React task manager app with:
- Add task functionality
- Mark tasks as complete
- Filter tasks by All, Active, Completed
- Clean and minimal UI

**Live URL:** https://your-vibe-link.netlify.app  

---

## Notes

- The tool generated a complete working application with UI and logic in a single step.
- The initial setup was extremely fast compared to manual coding.
- Some logic was spread across multiple files, making it harder to follow.
- Certain functions required additional reading to understand their behavior.
- Modifying features (like filter logic) was slightly difficult due to unfamiliar structure.
- Overall, it was great for quickly generating a working prototype.

---

## Key Takeaway

This approach is ideal for rapid prototyping, but the generated code can be harder to understand and maintain compared to manually built solutions.
